# 🏠 AI House Price Prediction

An end-to-end Machine Learning web application that predicts Bangalore house prices based on property details using Linear Regression.

---

## 🚀 Live Features

- Predict house prices instantly
- Modern responsive UI
- Machine Learning powered prediction
- Flask backend
- Trained on Bangalore House Price Dataset

---

## 🛠️ Tech Stack

- Python
- Flask
- Pandas
- NumPy
- Scikit-learn
- HTML5
- CSS3
- JavaScript

---

## 📊 Machine Learning Model

Model: Linear Regression

Performance

- R² Score: **0.797**
- MAE: **19.87 Lakhs**
- RMSE: **35.06 Lakhs**

---

## 📂 Dataset

Bengaluru House Price Dataset

Features used:

- Location
- Total Square Feet
- Bathrooms
- Balcony
- BHK

Target:

- House Price (Lakhs)
